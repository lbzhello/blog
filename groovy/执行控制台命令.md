## excute

```groovy
def sout = new StringBuilder(), serr = new StringBuilder()
def proc = 'ls /badDir'.execute()
proc.consumeProcessOutput(sout, serr)
proc.waitForOrKill(1000)
println "out> $sout err> $serr"
```

## 示例 -- 显示 svn 提交记录

```groovy
import reactor.core.publisher.Flux

def process = 'svn log -v -r "{2019-12-27T00:00:00}:{2019-12-27T23:59:59}" https://192.0.0.122/SecurityDetection/trunk/V2.x/src/components/scm/V1.0.x/scm-web'.execute()

def sep = '------------------------------------------------------------------------'

process.waitFor()

def logs = process.getInputStream().getText("gbk").split(sep)

Flux.fromArray(logs).map {
    println(it)
    it
}.subscribe()
```
